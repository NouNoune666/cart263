A new way to start your code

window.onload

Similar to p5.js's setup(), this ensures code runs only after all page resources (stylesheets, scripts, images) have loaded. This prevents trying to access elements before they exist in the DOM.

The DOM (Document Object Model)

Each object its own properties and methods Due to the existence of the DOM - we can use JavaScript to manipulate and alter the HTML elements in an HTML page.
